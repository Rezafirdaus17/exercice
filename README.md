## Tech
1. Python (Flask)

## How to run
1. Create your virtual env or used your global env and actived your env
   ```
    your_path_env/Scripts
    activate
   ```
2. Install lib
   ```
    pip install -r requirements.txt
   ```
3. run migrate
   ```
   flask db upgrade
   ```
4. run pytest
   ```
   python -m pytest
   ```
5. run server
   ```
   flask run
   ```
```
   it should be run on port 5000
```
